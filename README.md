# Customer Shopping Trend Analysis

## Project Overview
Analyzed 3,900+ customer transactions to uncover shopping patterns and provide business insights using Python.

## Questions Answered
1. Which product categories sell most in each season?
2. Does discount/promo code usage affect purchase amount?
3. How does age affect seasonal spending?
4. Which payment methods lead to higher spending?
5. How does purchase frequency affect average spending?
6. Which season has highest sales and average order value?

## Technologies Used
- Python (Pandas, NumPy)
- Visualization (Matplotlib, Seaborn)

## Key Insights
- **Clothing** dominates sales across all seasons, with Spring being the peak season
- Discounts and promo codes have **minimal to no impact** on both purchase amount and customer ratings
- **Young adults (18-29)** spend the most, particularly during the Fall season
- **Venmo and Credit Card** users tend to spend more compared to other payment methods
- **Less frequent shoppers** tend to have higher average spending per transaction
- **Fall season** leads in both total sales and average order value
